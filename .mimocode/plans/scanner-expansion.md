# План расширения сканера Telegram каналов

## Текущее состояние
- `smart_poster.py` — ищет группы по ключевым словам, постит промо (2-3 группы за запуск)
- `rent_scanner/` — мониторит каналы, отправляет алерты подписчикам
- Сканирование: по расписанию через APScheduler (раз в час)

## Цель
Расширить сканер чтобы он:
1. Сканировал все объявления на каналах каждые 60 минут
2. Автоматически подписывался на каналы с релевантными объявлениями
3. Отправлял алерты подписчикам при нахождении подходящих объявлений

## Задачи

### Задача 1: Расширение базы каналов
- Добавить каналы по городам: Berlin, Munich, Hamburg, Vienna, Amsterdam, London
- Каждый город: 3-5 каналов по аренде
- Формат: `{"name": "Berlin Apartments", "url": "https://t.me/...", "city": "berlin"}`

### Задача 2: Автоподписка на каналы
- При сканировании: проверять подписан ли бот на канал
- Если канал релевантный (содержит объявления об аренде) — подписываться
- Лимит: 5 подписок в день (чтобы не забанили)

### Задача 3: Сканирование сообщений
- Читать последние 10-15 сообщений с канала
- Искать ключевые слова: цена, площадь, город
- Извлекать URL объявлений
- Сохранять в Supabase (WebListings)

### Задача 4: Алерты подписчикам
- Сверять найденные объявления с подписками пользователей
- Отправлять в лС если город и цена совпадают
- Дедупликация через Supabase (WebListings)

### Задача 5: Интеграция с scheduler.py
- Запуск сканера раз в час через APScheduler
- Логирование результатов
- Обработка ошибок

## Структура файлов
```
rent_scanner/
  __init__.py
  __main__.py
  app.py          — основной класс RentScanner
  config.py       — конфигурация
  filters.py      — фильтры по ключевым словам
  formatting.py   — форматирование алертов
  sources.py      — список каналов-источников
  storage.py      — хранение данных (Supabase)
  channels.json   — расширенный список каналов
```

## Интеграция с scheduler.py
```python
# В scheduler.py добавить:
apscheduler.add_job(
    lambda: _run_async(run_channel_scanner()),
    CronTrigger(minute=0),  # Каждый час
    name="channel_scanner"
)
```
